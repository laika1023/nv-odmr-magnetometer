# Build log

One markdown file per month (`2026-07.md`, `2026-08.md`, ...). Newest entry at the TOP of each file.
Rules that make this work:
1. Write the entry BEFORE leaving the bench — 10 minutes max, use the template below.
2. Failures get logged with the same care as successes (they're the credible part).
3. Every photo referenced lives in `../media/` with the naming convention `YYYY-MM-DD_shortdesc.jpg`.
4. Decisions with lasting consequences also get a one-liner in `../docs/DECISIONS.md`.
