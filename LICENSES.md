# Licensing

This project uses a split license, standard for open-source hardware:

- **Code** (`firmware/`, `analysis/`): MIT License — see `LICENSE-MIT`.
- **Hardware designs & documentation** (`hardware/`, `report/`, `log/`, `docs/`): CC-BY-4.0.
  (Alternative for hardware: CERN-OHL-P v2 — download text from https://ohwr.org/cern_ohl_p_v2.txt and add as `LICENSE-CERN-OHL-P` if preferred.)

Rationale: permissive licenses maximize educational reuse, which is the stated purpose of this project (and the framing of its grant applications).
